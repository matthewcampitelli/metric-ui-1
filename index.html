Stepped Progress Control Component
Basic demonstration of display element used to provide visual indicator of level within a given range.

Give it a spin! Slide the position of the progress bar to adjust the current level of the metric, the bar progress indicator below will reflect the new value.

Effect implemented in using third party GSAP animation library (awesomeness) using ES6 classes so that the effect can be applied to different UI components; i.e. variables such as number of display elements and starting and ending values of range may be dynamically assigned.
